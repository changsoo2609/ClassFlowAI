# ClassFlowAI 기능 기준 목록

이 문서는 현재 코드와 배포 안내에서 확인되는 기능만 기록한다.

## 캡처와 조작

- `Ctrl + Shift + S`로 Windows 화면 캡처 실행
- 클립보드에 들어온 새 이미지 자동 감지
- 동일 이미지 해시를 이용한 중복 감지 지원
- 캡처 원본을 시간 기반 파일명으로 저장
- 휠클릭으로 OCR/CAP 모드 전환
- `Ctrl + 휠클릭`으로 감지 일시정지·재개
- `Shift + 휠클릭`으로 메인 창 표시·최소화
- 설정창에서 단축키 변경

## OCR 모드

- NVIDIA OCR 모델을 이용한 이미지 문자 추출
- 이미지 확대, 최대 크기 제한과 전처리 옵션
- OCR 결과 후처리
- 처리 결과 화면 표시
- OCR 완료 후 텍스트 자동 복사
- 원본 이미지 기준 OCR 보정
- 보정 OCR 자동 복사
- 원본 이미지와 보정 OCR을 함께 사용한 내용 해석
- OCR/CAP 모델 ID와 모델 페이지 URL 설정

## CAP 모드

- 이미지 전체의 의미와 구조를 추론 모델로 해석
- CAP 결과를 결과창에 표시
- 처리 완료 후 원본 이미지 클립보드 유지
- 선택한 CAP 캡처의 원본 이미지를 클립보드에 다시 복사
- 사용자가 요청할 때만 CAP 해석 텍스트 복사
- CAP 프롬프트 편집

## 수업 기록 관리

- 현재 수업의 기록과 원본 이미지를 보존한 채 새 수업 시작
- 자동 생성된 독립 수업 폴더 또는 기존 저장 폴더에서 이전 수업 열기
- 마지막으로 연 수업을 저장 위치별로 기억하고 다음 실행 시 복원
- 캡처 순서와 처리 결과 기록
- 이전·다음 캡처 탐색
- 현재 캡처 삭제
- 현재 기록과 캡처 파일의 정합성 보정
- 기록을 기준으로 출력 문서 재생성
- 현재 수업 초기화 시 기록만 초기화하거나 원본 이미지까지 삭제하도록 선택
- 유지하거나 삭제하지 못한 이미지가 초기화 뒤 목록으로 자동 복원되지 않도록 처리
- 캡처 저장 폴더 열기
- 사용자 지정 저장 위치와 날짜별 폴더 사용

## 내보내기

- 시간순 캡처 흐름 Markdown 생성
- OCR/CAP 결과 타임라인 생성
- 이미지가 포함된 HTML 흐름 미리보기
- 원본 이미지, 시간순 기록, 안내문과 프롬프트를 포함한 GPT 전달 ZIP 생성
- GPT ZIP 프롬프트 편집

## 학습카드 가져오기와 검토

- `study_cards.json` 스키마와 카드별 필수 구조 검사
- 카드 유형, 정답 상태, 난이도와 검토 필요 상태 검사
- 근거 이미지 누락과 내용 품질 경고
- 공백·대소문자·문장부호를 정규화한 동일·유사 질문 중복 탐지
- 학습 단위당 카드 3장 초과 가능성 경고
- JSON을 수정하지 않는 Python API와 명령행 검증 도구 제공
- `study_cards.json` 또는 `notion_paste_package.zip` 가져오기
- ZIP 경로 탈출을 차단하고 카드가 참조하는 이미지만 추출
- 기존 사용자 수정과 검토 상태를 보존하는 중복 병합
- 동일 이름 이미지의 내용 해시 비교와 충돌 시 안전한 이름 분리
- 수업별 카드, 가져오기 이력과 근거 이미지 저장
- 메인 창의 `학습카드` 버튼과 별도 검토 창
- 상태·과목·카드 유형 필터, 질문·답·핵심 요소·태그 수정
- 카드 승인, 제외, 검토 대기 전환과 근거 이미지 열기
- 삭제 대신 제외 상태를 사용해 원본 카드 정보 보존

## UI와 상태 표시

- 메인 Tkinter 창
- 현재 모드, 캡처 순서, 처리 결과와 실행 시간 표시
- OCR/CAP 결과에 맞는 작업 버튼 활성화
- 드래그 가능한 미니 상태창
- OCR 대기, CAP 대기, 진행, 완료, 실패, 일시정지 상태별 색상 표시
- 캡처 시 메인 창 숨김 및 복원 옵션
- 실행 준비 진행률을 CMD에 표시

- 사용자 API 키 저장
- 환경변수 API 키 확인 지원
- 모델, 요청 시간 제한, 재시도, 이미지 처리 옵션 설정
- 설정 JSON 원자적 저장
- 시작 성공 플래그 생성
- 시작 실패 상세 로그 기록
- Python·Tkinter·필수 패키지 설치 및 실행 전 검증

## 현재 기준에서 별도 검증이 필요한 항목

다음은 코드 경로가 존재하지만 실제 Windows 환경과 API 연결을 이용한 종단 검증이 필요한 기능이다.

- 변경 가능한 전역 단축키 조합 전체
- OCR 모델 실제 연결과 응답 형식
- CAP 모델 실제 연결과 재시도·시간 제한
- 캡처 20~50장 기록의 장시간 안정성
- GPT ZIP을 이용한 `notion_paste_package.zip` 생성 과정
- Windows CF_HTML 방식의 최종 Notion 붙여넣기
- 실제 수업·정보처리기사 캡처에서 생성된 `study_cards.json` 품질 검증
- 실제 결과물을 이용한 카드 가져오기·사용자 검토 흐름
- 망각곡선 복습 일정과 파인만 답변 평가
