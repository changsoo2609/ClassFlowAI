# ClassFlowAI 기능 기준 목록

이 문서는 현재 코드와 배포 안내에서 확인되는 기능만 기록한다.

## 캡처와 조작

- `Ctrl + Shift + S`로 Windows 화면 캡처 실행
- 클립보드에 들어온 새 이미지 자동 감지
- 동일 이미지 해시를 이용한 중복 감지 지원
- 캡처 원본을 시간 기반 파일명으로 저장
- 현재 수업 캡처 목록 또는 미리보기 우클릭 메뉴에서 OCR/CAP 원본 이미지 다시 복사
- 휠클릭으로 OCR/CAP 모드 전환
- `Ctrl + 휠클릭`으로 감지 일시정지·재개
- `Shift + 휠클릭`으로 메인 창 표시·최소화
- 미니 상태창 우클릭으로 메인 창 열기, 감지 일시정지·재개와 확인 후 종료
- 설정창에서 단축키 변경

## OCR 모드

- NVIDIA OCR 모델을 이용한 이미지 문자 추출
- 이미지 확대, 최대 크기 제한과 전처리 옵션
- OCR 결과 후처리
- 처리 결과 화면 표시
- OCR 완료 후 텍스트 자동 복사
- 원본 이미지 기준 OCR 보정
- 보정 OCR 자동 복사
- 원본 이미지와 보정 OCR을 함께 사용한 내용 해석
- OCR/CAP 모델 ID와 모델 페이지 URL 설정

## CAP 모드

- 이미지 전체의 의미와 구조를 추론 모델로 해석
- CAP 결과를 결과창에 표시
- 처리 완료 후 원본 이미지 클립보드 유지
- 선택한 CAP 캡처의 원본 이미지를 클립보드에 다시 복사
- 사용자가 요청할 때만 CAP 해석 텍스트 복사
- CAP 프롬프트 편집

## 수업 기록 관리

- 현재 수업의 기록과 원본 이미지를 보존한 채 새 수업 시작
- 자동 생성된 독립 수업 폴더 또는 기존 저장 폴더에서 이전 수업 열기
- 마지막으로 연 수업을 저장 위치별로 기억하고 다음 실행 시 복원
- 캡처 순서와 처리 결과 기록
- 현재 수업 캡처 목록에서 사용자 학습 흐름 순서를 위·아래로 조정하고 촬영 순서로 복원
- 파일명·촬영 시각·캡처 ID·OCR/CAP 결과와 분리된 `display_order`만 변경
- 이전·다음 캡처 탐색
- 현재 캡처 삭제
- 현재 기록과 캡처 파일의 정합성 보정
- 기록을 기준으로 출력 문서 재생성
- 현재 수업 초기화 시 기록만 초기화하거나 원본 이미지까지 삭제하도록 선택
- 유지하거나 삭제하지 못한 이미지가 초기화 뒤 목록으로 자동 복원되지 않도록 처리
- 캡처 저장 폴더 열기
- 사용자 지정 저장 위치와 날짜별 폴더 사용
- 새 수업을 `실시간저장\날짜\시간` 구조로 저장하고 기존 수업 경로를 읽기 호환
- 화면에는 `실시간저장`부터 시작하는 짧은 저장 위치 표시
- 캡처 삭제 시 수업 폴더 내부 원본·전용 보조 파일과 학습 참조 정리
- 수업 외부 파일 삭제 차단과 원본 삭제 실패 시 레코드 유지

## 공통 수업 흐름과 항목별 복사

- OCR/CAP 결과를 동일한 `FlowDocument` 구조로 정규화
- 사용자 `display_order`와 연속된 관련 캡처 묶음 유지
- 캡처 바로 뒤에 관련 해설·코드·메모 배치
- 그림과 텍스트·코드를 구분한 단일 수업 흐름 화면
- 그림 우클릭 시 재압축하지 않은 원본 이미지를 Windows 이미지 클립보드로 복사
- 해설·메모·코드 우클릭 시 선택한 블록만 일반 텍스트로 복사
- OCR 텍스트를 먼저 표시한 뒤 수업 흐름용 해석을 백그라운드에서 갱신
- 별도 작성 탭이나 전체 복사 버튼 없이 항목별 복사 흐름에 집중
- 구조화 결과를 수업별 `state/flow_document.json`에 저장

## UI와 상태 표시

- 메인 Tkinter 창
- 현재 모드, 캡처 순서, 처리 결과와 실행 시간 표시
- OCR/CAP 결과에 맞는 작업 버튼 활성화
- 드래그 가능한 미니 상태창
- OCR 대기, CAP 대기, 진행, 완료, 실패, 일시정지 상태별 색상 표시
- 캡처 시 메인 창 숨김 및 복원 옵션
- 실행 준비 진행률을 CMD에 표시

## 설정과 실행 안정성

- 사용자 API 키 저장
- 환경변수 API 키 확인 지원
- 모델, 요청 시간 제한, 재시도, 이미지 처리 옵션 설정
- 설정 JSON 원자적 저장
- 시작 성공 플래그 생성
- 시작 실패 상세 로그 기록
- Python·Tkinter·필수 패키지 설치 및 실행 전 검증

## 현재 기준에서 별도 검증이 필요한 항목

다음은 코드 경로가 존재하지만 실제 Windows 환경과 API 연결을 이용한 종단 검증이 필요한 기능이다.

- 변경 가능한 전역 단축키 조합 전체
- OCR 모델 실제 연결과 응답 형식
- CAP 모델 실제 연결과 재시도·시간 제한
- 캡처 20~50장 기록의 장시간 안정성
